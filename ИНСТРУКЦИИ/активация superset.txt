чтобы не было 500 ошибки, нужно зарегистрировать пользователя
открываем терминал (cmd/powershell/bash) в той же папке, где находится docker-compose.yml ("главная" папка)
запускаем контейнер superset

docker exec -it superset bash

создаем пользователя (админа)

superset fab create-admin --username admin --firstname Admin --lastname User --email admin@admin.com --password admin

инициализируем бд и миграции

superset db upgrade
superset init

выходим

exit

перезапускаем контейнер

docker restart superset

теперь superset работает. чтобы открыть в браузере введи
http://localhost:8088/
в открывшемся окне, если попросит, введи admin дважды

чтобы подключить ClickHouse в Superset в верхнем правом углу находим +, далее:
Data -> Connect database -> Choose a database... -> ClickHouse
в открывшемся окне в SQLAlchemy URI вводим и нажимаем Connect

clickhouse+http://default:@clickhouse:8123/default
